# Deke Sharon Website Redesign Plan

## Executive Summary

Transform the current professional/austere website into an inviting, playful, and energetic experience inspired by the visual language of "Glee" — capturing the joy and community spirit of a cappella music while serving as an effective lead generation tool for Deke Sharon's services.

---

## 1. Target Audience

### Primary Audiences
1. **A Cappella Singers & Directors**
   - High school students and advisors
   - Collegiate group members and music directors
   - Community group singers and organizers

2. **Performing Arts Faculty**
   - Private college/university administrators
   - Recruitment and admissions officers
   - Music department chairs

### Audience Needs
- Quick understanding of available services
- Social proof through testimonials and credentials
- Easy booking/inquiry process
- Mobile-first experience (checking between classes, on-the-go)

### 3.2 Typography

#### Font Pairing

| Role | Font Family | Weight | Usage |
|------|-------------|--------|-------|
| **Display/Hero** | **Fredoka One** | 400 | Hero headlines, impact statements |
| **Headings** | **Nunito** | 700, 800 | Section headings, card titles |
| **Body** | **Nunito** | 400, 600 | Body text, descriptions |
| **Accent** | **Permanent Marker** | 400 | Playful callouts, quotes (sparingly) |

**Rationale:**
- **Fredoka One**: Rounded, friendly, playful — captures "Glee" energy without being childish
- **Nunito**: Highly readable, warm, approachable — works across all sizes
- **Permanent Marker**: Hand-drawn feel for special emphasis (use sparingly)

#### Type Scale (Mobile-First)

```css
/* Base: 16px */
--text-xs: 0.75rem;    /* 12px - captions */
--text-sm: 0.875rem;   /* 14px - small text */
--text-base: 1rem;     /* 16px - body */
--text-lg: 1.125rem;   /* 18px - lead text */
--text-xl: 1.25rem;    /* 20px - card titles */
--text-2xl: 1.5rem;    /* 24px - section heads */
--text-3xl: 1.875rem;  /* 30px - page titles */
--text-4xl: 2.25rem;   /* 36px - hero mobile */
--text-5xl: 3rem;      /* 48px - hero tablet */
--text-6xl: 3.75rem;   /* 60px - hero desktop */
```

### 3.3 Image Style Guidelines

#### Photography Style
- **Bright and well-lit**: No dark, moody shots
- **Action shots**: Groups singing, workshops in progress, joyful expressions
- **Authentic moments**: Candid over posed
- **Diverse representation**: Various ages, backgrounds, group types
- **Color saturation**: Slightly boosted for vibrancy

#### Illustration Style
- **Flat, vector-based** illustrations
- **Bright color palette** matching brand colors
- **Musical motifs**: Notes, sound waves, microphones
- **Character-friendly**: Simple, joyful figures when appropriate

#### Iconography
- **Style**: Rounded, filled icons (not thin outlines)
- **Library**: Lucide or Phosphor Icons (rounded variants)
- **Color**: Use accent colors, not just gray

#### Decorative Elements
- **Confetti patterns**: Subtle background accents
- **Sound wave graphics**: Abstract musical representations
- **Geometric shapes**: Circles, rounded rectangles
- **Gradient overlays**: On hero images

## 4. Mobile-First Responsive Strategy

### Breakpoints

```css
/* Mobile First */
--breakpoint-sm: 640px;   /* Large phones */
--breakpoint-md: 768px;   /* Tablets */
--breakpoint-lg: 1024px;  /* Small laptops */
--breakpoint-xl: 1280px;  /* Desktops */
--breakpoint-2xl: 1536px; /* Large screens */
```

### Mobile Considerations

#### Navigation
- Hamburger menu with full-screen overlay
- Large touch targets (min 48px)
- Bottom-positioned key CTAs
- Sticky header with scroll-triggered shrink

#### Touch Interactions
- Swipeable testimonial carousels
- Tap-to-expand service cards
- Pull-to-refresh on list pages
- No hover-dependent interactions

#### Performance
- Lazy load images below fold
- Skeleton loaders for dynamic content
- Optimized image formats (WebP, AVIF)
- Critical CSS inlining

#### Typography Scaling
- Fluid typography using clamp()
- Reduced heading sizes on mobile
- Increased line height for readability

```css
/* Example fluid heading */
h1 {
  font-size: clamp(2.25rem, 5vw + 1rem, 3.75rem);
}
```

## 6. Content Tone & Voice

### Brand Voice Attributes
- **Enthusiastic** but not over-the-top
- **Knowledgeable** but not condescending
- **Warm** and approachable
- **Encouraging** and supportive
- **Playful** with moments of humor

### Writing Guidelines
- Use "you" and "your" frequently
- Lead with benefits, not features
- Short paragraphs (2-3 sentences max)
- Active voice
- Avoid jargon unless audience-appropriate
- Include personality in headlines

### Example Headlines
- ❌ "Professional A Cappella Services"
- ✅ "Let's Make Your Group Unforgettable"

- ❌ "Arrangement Services"
- ✅ "Arrangements That Make Voices Soar"
