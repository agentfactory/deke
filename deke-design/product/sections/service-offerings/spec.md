# Service Offerings Specification

## Overview
Public-facing pages that showcase available coaching services: Speaking, Coaching, Workshops, Masterclass, and Arrangements. Visitors can explore service details and request bookings.

## User Flows
- View list of all service offerings with pricing and key details
- Filter services by type (speaking, coaching, workshops, masterclass, arrangements)
- Click a service to view full details including features and testimonials
- Request booking for a specific service
- Contact coach with questions about services

## UI Requirements
- Showcase view displaying all service offerings in cards
- Each service card shows: title, type, pricing, duration, brief description, featured image
- Service detail includes: full description, what's included, pricing tiers, booking CTA
- Mobile-responsive design optimized for public visitors
- Visual hierarchy emphasizing the value proposition of each service
- Booking request button prominently displayed on each service
