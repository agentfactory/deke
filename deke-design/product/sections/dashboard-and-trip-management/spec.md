# Dashboard & Trip Management Specification

## Overview
A mobile-optimized dashboard for managing trips, bookings, expenses, and tracking profitability across all coaching engagements.

## User Flows
- View list of all trips with key metrics (dates, location, bookings count, revenue/expenses, status)
- Create a new trip by specifying location and dates
- Click a trip to view full details (bookings, expenses, participants, profitability breakdown)
- Add bookings to a trip
- Track and categorize expenses for each trip
- View profitability calculations showing revenue vs expenses

## UI Requirements
- List view as primary interface showing all trips
- Each trip card displays: dates, location, number of bookings, revenue summary, expenses summary, trip status
- Detail view page for individual trips with complete breakdown
- Actions: Create trip button, Add booking, Add expense
- Mobile-responsive design optimized for on-the-go management
