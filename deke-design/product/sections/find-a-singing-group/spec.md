# Find a Singing Group Specification

## Overview
A public-facing service where people request help finding local singing groups based on age, location, and music preferences. Coaches use this admin interface to view requests, match them with local groups, and respond personally.

## User Flows
- View list of all group requests with key details (name, location, preferences, match status)
- Filter requests by location, age range, and music preferences
- View individual request details including full preferences and contact information
- Mark requests as matched, in-progress, or responded
- Search for requests by name or location
- View match score indicators showing potential group fits

## UI Requirements
- List view as primary interface showing all group requests
- Each request card displays: requester name, age, location, music preferences, match status, submitted date
- Filter controls for location, age range, and music preferences
- Search bar for quick lookups
- Match status indicators (new, in-progress, matched, responded)
- Mobile-responsive design for reviewing requests on-the-go
